## Image Guidelines

1. **Optimize all images** before uploading
2. Use **WebP format** where possible for better performance
3. Provide **responsive versions** for different screen sizes
4. Include **alt text** in all image references
5. Recommended tools: TinyPNG, Squoosh, ImageOptim
